# Driving Theory Test Application

![Description of the image](./image.jpg)

This project is an application designed to help users practice for driving license tests. The questions come from the official website of the Polish government [gov.pl](https://www.gov.pl/web/infrastruktura/prawo-jazdy), and they are the same as those used in the official driving exams.

## Features

- ✅ **Mobile-Friendly:** The application is designed to work seamlessly on mobile devices, ensuring you can practice anytime, anywhere.

- ✅ **Interactive Tests:** Users can take multiple-choice tests that simulate real driving license exams.
- ✅ **Official Questions:** All test questions are sourced from the official Polish government website, ensuring they are identical to those used in the official driving exams.
- ✅ **Instant Feedback:** After completing each test, users receive feedback on their answers to help them learn.
- ⚠️ **User Accounts (Future):** Users will be able to create accounts and save their progress (coming soon).
- ⚠️ **Progressive Web App (PWA) Support (Future):** In the future, the app may evolve into a full PWA, allowing offline access and push notifications.
